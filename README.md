⭐ If you find this useful, please star the repo — it helps a lot!

Track large wallet movements (whales) across multiple chains and detect major transactions in real time.

Built with **Next.js, TypeScript, viem, Moralis/Alchemy, and Tailwind CSS**.

## ✨ Features

- 🐋 Track predefined whale wallets  
- 🔍 Monitor large transactions in real-time (polling-based)  
- 📊 View recent whale activity  
- 💰 Highlight biggest inflows and outflows  
- 🌐 Multi-chain support (Ethereum, Polygon, Base,)  
- ⚡ Fast and responsive UI  
- 🔔 Optional alerts (UI / Telegram — upcoming)  

---

## 💡 Why This Project

Crypto markets are heavily influenced by **whale movements**.

But most tools:
- are slow  
- cluttered  
- not developer-friendly  

This dashboard provides a **clean and real-time view of large wallet activity**, helping users:
- detect big moves early  
- analyze market behavior  
- build trading insights  

---

## 🧠 How It Works

1. Track a list of whale wallets  
2. Fetch transaction data using APIs  
3. Filter large transactions  
4. Calculate:
   - inflow vs outflow  
   - transaction size  
5. Display insights via dashboard UI

## ⚙️ Setup

### 1. Clone the repo

```bash
git clone https://github.com/YOUR_USERNAME/whale-tracker-dashboard.git
cd whale-tracker-dashboard
```
### 2. Install dependencies

```bash
npm install
```

### 3. Add environment variables

###### Create a .env.local file:

```bash
MORALIS_API_KEY=your_api_key_here
ALCHEMY_API_KEY=your_api_key_here
```


## 🔑 How to get API keys

- Moralis
- Go to https://moralis.io/
- Sign up / log in
- Create a project
- Copy your API key
- Alchemy (optional)
- Go to https://www.alchemy.com/
- Create an app
- Copy your API key

### 4. Run the app

```bash
npm run dev
```

### Open:

```bash
http://localhost:3000
```
